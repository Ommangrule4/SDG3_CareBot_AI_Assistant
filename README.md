CareBot: AI-Powered Health Assistant (SDG 3)
Project Overview
CareBot is an AI-driven conversational platform built on Botpress Studio. It acts as a first-line virtual health navigator designed to promote United Nations Sustainable Development Goal 3: Good Health and Well-Being.

Core Features
Empathetic AI Triage Engine: Analyzes natural language symptom inputs dynamically.

Safe Guidance & Disclaimers: Provides basic home care benchmarks alongside strict medical emergency flags.

Resource Routing: Links users directly to established medical data hubs like the WHO and Mayo Clinic.
